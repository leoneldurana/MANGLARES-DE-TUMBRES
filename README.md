# MANGLARES-DE-TUMBRES
Los manglares de Tumbes son uno de los sumideros de carbono más eficientes del planeta. El objetivo de este proyecto es utilizar herramientas tecnológicas (como teledetección, machine learning y procesamiento de datos geoespaciales) para cuantificar este almacenamiento y apoyar los esfuerzos de conservación y mitigación del cambio climático.
